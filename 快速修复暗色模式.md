# 快速修复暗色模式

## 问题
只有滚动条区域变色，其他地方不变。

## 原因
Tailwind v4 需要在 CSS 中明确定义 dark variant。

## 已应用的修复

### app/globals.css
```css
@import "tailwindcss";

@variant dark (&:where(.dark, .dark *));
```

## 立即执行

### 步骤 1: 停止开发服务器
按 `Ctrl + C` 停止当前运行的服务器

### 步骤 2: 清除缓存并重新构建
```bash
Remove-Item -Recurse -Force .next -ErrorAction SilentlyContinue
npm run build
```

### 步骤 3: 启动开发服务器
```bash
npm run dev
```

### 步骤 4: 强制刷新浏览器
- Windows: `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

### 步骤 5: 测试
点击主题切换按钮，整个页面应该变色。

## 预期效果

### 切换到暗模式时
- 整个页面背景变成深色
- 所有文字变成浅色
- 卡片和组件都变成深色主题
- 不仅仅是滚动条

### 切换到亮模式时
- 整个页面背景变成白色
- 所有文字变成深色
- 卡片和组件都变成亮色主题

## 如果还是不工作

请在浏览器控制台执行：
```javascript
// 手动添加 dark class
document.documentElement.classList.add('dark')

// 等待 1 秒后检查
setTimeout(() => {
  const body = document.querySelector('body')
  const bg = window.getComputedStyle(body).backgroundColor
  console.log('Body background:', bg)
  
  const h1 = document.querySelector('h1')
  const color = window.getComputedStyle(h1).color
  console.log('H1 color:', color)
}, 1000)
```

然后告诉我输出的结果。
