# Markdown 写作指南

Markdown 是一种轻量级标记语言，让你可以使用纯文本格式编写文档，然后转换成格式丰富的 HTML 页面。

## 基本语法

### 标题

使用 `#` 符号创建标题，数量表示标题级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式

- **粗体文本**：使用 `**文本**` 或 `__文本__`
- *斜体文本*：使用 `*文本*` 或 `_文本_`
- ~~删除线~~：使用 `~~文本~~`

### 列表

无序列表：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3

有序列表：

1. 第一项
2. 第二项
3. 第三项

### 链接和图片

链接：[访问 GitHub](https://github.com)

图片：`![替代文本](图片URL)`

### 代码

行内代码：使用 `代码` 包裹。

代码块：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

## 写作建议

1. **保持简洁**：使用简短的段落和句子
2. **使用标题**：合理组织内容结构
3. **添加代码示例**：技术文章中包含实际代码
4. **使用列表**：让信息更易于阅读
5. **添加图片**：适当的视觉元素能增强理解

## 总结

Markdown 是一个强大而简单的写作工具。掌握这些基本语法后，你就可以开始创作优质的博客内容了！

Happy writing! 🎉
