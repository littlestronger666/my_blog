## 📋 配置所需信息

你需要从 https://giscus.app/zh-CN 获取 4 个值：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `repo` | 你的 GitHub 仓库 | `zhangsan/my-blog` |
| `repoId` | 仓库 ID | `R_kgDOL1234567` |
| `category` | Discussion 分类 | `Announcements` |
| `categoryId` | 分类 ID | `DIC_kwDOL1234567` |

## 🔧 配置步骤概览

### 1. 准备 GitHub 仓库
- 创建或使用现有的公开仓库
- 启用 Discussions 功能

### 2. 安装 Giscus App
- 访问 https://github.com/apps/giscus
- 安装到你的仓库

### 3. 获取配置信息
- 访问 https://giscus.app/zh-CN
- 输入仓库信息
- 复制生成的配置

### 4. 更新代码
- 打开 `components/Comments.tsx`
- 替换 4 个配置值
- 保存文件

### 5. 测试
```bash
npm run dev
```
访问文章页面，查看评论框
