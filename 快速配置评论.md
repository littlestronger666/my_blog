# 快速配置评论系统

## 🚀 5 分钟快速配置

### 1️⃣ 准备 GitHub 仓库（2分钟）

1. 登录 GitHub
2. 创建公开仓库或使用现有仓库
3. 进入仓库 Settings → Features
4. 勾选 **Discussions**

### 2️⃣ 安装 Giscus（1分钟）

1. 访问：https://github.com/apps/giscus
2. 点击 **Install**
3. 选择你的博客仓库
4. 点击 **Install**

### 3️⃣ 获取配置（1分钟）

1. 访问：https://giscus.app/zh-CN
2. 输入仓库：`你的用户名/仓库名`
3. 选择分类：**Announcements**
4. 复制页面底部显示的 4 个值：
   - `data-repo`
   - `data-repo-id`
   - `data-category`
   - `data-category-id`

### 4️⃣ 更新配置（1分钟）

打开 `components/Comments.tsx`，替换这 4 行：

```tsx
repo="YOUR_GITHUB_USERNAME/YOUR_REPO_NAME"  // 改成你的
repoId="YOUR_REPO_ID"                       // 改成你的
category="Announcements"                     // 改成你的（如果不同）
categoryId="YOUR_CATEGORY_ID"               // 改成你的
```

**示例：**
```tsx
repo="zhangsan/my-blog"
repoId="R_kgDOL1234567"
category="Announcements"
categoryId="DIC_kwDOL1234567"
```

### 5️⃣ 测试（30秒）

```bash
npm run dev
```

访问任意文章页面，滚动到底部，应该能看到评论框！

## 📋 需要的信息

从 https://giscus.app/zh-CN 获取：

| 需要替换的 | 在哪里找 | 示例 |
|-----------|---------|------|
| `repo` | 你的仓库地址 | `zhangsan/my-blog` |
| `repoId` | giscus.app 生成 | `R_kgDOL1234567` |
| `category` | 你选择的分类 | `Announcements` |
| `categoryId` | giscus.app 生成 | `DIC_kwDOL1234567` |

## ⚠️ 重要提示

1. **仓库必须是公开的（Public）**
2. **必须启用 Discussions 功能**
3. **必须安装 Giscus App**
4. **配置信息必须完全正确**
